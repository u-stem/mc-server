# 統合版（iOS / Android / Windows）

スマートフォン、タブレット、Windows PC からサーバーに接続する手順です。
これらのデバイスは直接サーバーを追加できます。

## 必要なもの

- Minecraft 統合版（Bedrock Edition）
- Tailscale（無料）- 外部ネットワークから接続する場合

## 対応デバイス

- iPhone / iPad（iOS）
- Android スマートフォン / タブレット
- Windows 10 / 11（Bedrock版）

## 同じネットワーク内での接続

ホストと同じ Wi-Fi / LAN に接続している場合は、Tailscale なしで接続できます。

### 手順

1. Minecraft を起動し「遊ぶ」をタップ/クリック
2. 「サーバー」タブを選択
3. 「サーバーを追加」をタップ/クリック
4. サーバー情報を入力:
   - サーバー名: 任意の名前
   - サーバーアドレス: ホストのローカルIPアドレス（例: `192.168.1.10`）
   - ポート: `19132`（またはホストから指定されたポート）
5. 「保存」してサーバーを選択

## 外部ネットワークからの接続

ホストと異なるネットワークから接続する場合は、Tailscale が必要です。

### 1. Tailscale をインストール

- iOS: App Store で「Tailscale」を検索してインストール
- Android: Google Play で「Tailscale」を検索してインストール
- Windows: https://tailscale.com/download からダウンロード

### 2. ホストからの招待を承諾

1. ホストから招待メールが届く
2. メール内の招待リンクをタップ
3. 「Accept invite」をタップして招待を承諾
4. Tailscale アプリで接続状態が「Connected」になっていることを確認

### 3. サーバーに接続

1. Tailscale が接続中であることを確認
2. Minecraft を起動し「遊ぶ」をタップ/クリック
3. 「サーバー」タブを選択
4. 「サーバーを追加」をタップ/クリック
5. サーバー情報を入力:
   - サーバー名: 任意の名前
   - サーバーアドレス: サーバーの Tailscale IP（例: `100.64.0.1`）
   - ポート: `19132`（またはホストから指定されたポート）
6. 「保存」してサーバーを選択

## 接続情報

ホストから以下の情報を受け取ってください:

| 項目 | 確認内容 |
|------|----------|
| サーバーアドレス | ローカルIP または Tailscale IP |
| ポート番号 | 通常は 19132（UDP） |
| サーバータイプ | Bedrock専用 または GeyserMC（Java版） |

## トラブルシューティング

### 接続できない

- サーバーが起動しているかホストに確認
- ポート番号が正しいか確認（デフォルト: 19132）
- Tailscale が「Connected」になっているか確認（外部接続の場合）

### 「世界に接続できませんでした」と表示される

- サーバーアドレスとポートが正しいか確認
- Tailscale アプリが起動しているか確認
- ホストに招待されているか確認

### GeyserMC サーバーでログインできない

- Floodgate 使用時は、プレイヤー名の前にプレフィックス（通常 `.`）が必要な場合があります
- ホワイトリストに `.あなたの名前` の形式で登録されているか確認してください
